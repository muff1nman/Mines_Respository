Step 1: Setup Git

For additional help and pictures take a look here.

For Linux
For Windows
For Mac

Linux:

Get the package git-core

Fedora:
sudo apt-get install git-core
Fedora:
sudo yum install git-core

Setup your personal info that you will be identified by in the respository

git config --global user.name "Put your name here"
git config --global user.email "Put your Mines email here"

Setup an SSH key

First, make sure you don't already have SSH keys by checking to see if the directory cd ~/.ssh does not exist. If not, in the terminal run:
ssh-keygen -t rsa -C "Comment to identify this key"

Add SSH public key to GitHub

Copy the contents of id_rsa.pub and paste them into gitHub under Account Settings > SSH Keys > Add SSH key where it says Key

Setup gitHub token

Go to github and click on account settings and then the tab account settings. Find the API token and using this token execute these commands
git config --global github.user Enter_your_github_username_here_without_quotations
git config --global github.token Enter_your_APItoken_here_"_"

Windows:

Install the latest Git distribution.

You can find an executable installer here.
Select the default options at every step. (Simply "Yes/Next" through everything)

Setup an SSH key

Open Git Bash

$ cd ~/.ssh
(Changes the current directory to your ssh folder. This should fail. If it doesn't, you already have a key and can skip this section.)

$ ssh-keygen -t rsa -C "Comment to identify this key"
(Your email makes a good identifying comment)

NOTE: When prompted for a file to save to, press enter without typing anything to choose the default.
NOTE: When prompted for a passphrase, choose something you can remember. Theoretically, you should choose something secure, but that's not a huge issue for our purposes.

Add SSH public key to GitHub

Open Git GUI
Choose Help --> SSH Key If you don't see a key displayed in the window, press Generate Key and click through. (You should have generated a key previously)
Copy the block of text to your clipboard. Do not change the whitespace of the text.
Open your github account, and go to Account Settings, then SSH Keys.
Click on Add SSH Key, then paste the block of text from earlier.

Configure Identifying Local Variables

Open Git Bash again.
Run the following:

$ git config --global user.name "Firstname Lastname"
(This will identify you on your commit tags)
$ git config --global user.email "your_email@youremail.com"
(This will allow other git users to contact you with questions about your commits)
$ git config --global github.user "username"
(This is your github login name)
$ git config --global github.token "0123456789yourf0123456789token"
(This can be found under your github Account Settings, under Account Settings on the left)

Step 2: Create Local Clone of the Repository

With Git GUI

Select Remote -> Add Remote from the menu.

Name: What you want to name the repository on your local machine. Recommended that you keep the repository name the same.
Location: URI of the remote repository. Should be of the form git@github.com:DirectoryAddressWithSomeSlashesMaybe.git

Unless you have reason not to, leave "Fetch Immediately" selected so you get all the files on the repository.

The repository will now be open in the GUI window. When you start the GUI again, you will have re-open the repository. (It will probably show up as a "Recent Repository," however)
