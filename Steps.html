Linux:

Get the package git-core

Ubuntu:
sudo apt-get install git-core
Fedora:
sudo yum install git-core

Setup your personal info that you will be identified by in the respository

git config --global user.name "Put your name here"
git config --global user.email "Put your Mines email here"

Setup an SSH key

In terminal run:
ssh-keygen -t rsa -C "Comment to identify this key"

Add SSH public key to GitHub

Copy the contents of id_rsa.pub and paste them into gitHub under Account Settings > SSH Keys > Add SSH key where it says Key

Setup gitHub token

Go to github and click on account settings and then the tab account settings. Find the API token and using this token execute these commands
git config --global github.user Enter_your_github_username_here_without_quotations
git config --global github.token Enter_your_APItoken_here_"_"
